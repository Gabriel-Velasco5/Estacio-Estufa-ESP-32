# Estacio-Estufa-ESP-32
Projeto feito com a plataforma Wokwi, utilizando a placa ESP-32 para fazer a conexão com o site e coletar informações da estufa, como temperatura, umidade e luminosidade, no site é possível alterar os valores, e quando esses valores estão fora da média, conseguimos estabilizar também pelo site através de servos motores que atuam como relés.

As bibliotecas utilizadas foram
LiquidCrystal I2C
DHT sensor library
ESP32Servo
